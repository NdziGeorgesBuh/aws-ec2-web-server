# AWS EC2 Web Server Deployment

## Overview
This project demonstrates how to deploy a web server on AWS EC2 and make it accessible over the internet.

## What I did
- Launched an EC2 instance (Amazon Linux)
- Configured security groups (SSH and HTTP access)
- Connected securely using SSH (.pem key)
- Installed Apache (httpd)
- Started and enabled the web server
- Deployed a live webpage accessible via public IP

## Technologies Used
- AWS EC2
- Linux (Amazon Linux 2023)
- Apache HTTP Server
- SSH

## Result
The web server is live and accessible via:
http://51.21.191.40
